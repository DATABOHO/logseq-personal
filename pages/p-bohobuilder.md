	- ### Quick Capture (Incoming Page)
		- Auto-focused text editor for immediate input
		- Screenshot paste detection from clipboard
		- Image preview with compression (70% JPEG)
		- Inline icon picker (20 SF Symbols)
		- Inline color picker (6 vibrant colors)
		- Category field for organization
		- Save & Clear functionality
	- ### Card Management
		- Create custom cards with text, images, icons, colors
		- Edit existing cards (pencil button in detail view)
		- Delete cards (swipe-to-delete or trash button with confirmation)
		- View cards in scrolling list with glassmorphism design
		- Detail view with full card information
	- ### Data Persistence
		- UserDefaults + Codable for automatic saving
		- Survives app restarts
		- Image storage as compressed Data
		- Category support for organization
	- ### Navigation
		- Tab-based navigation (TabView)
		- Incoming tab (default landing page)
		- Cards tab (saved collection)
		- Shared ViewModel between tabs
	- ### Meditation Timer
		- Configurable session duration (5-60 minutes)
		- Adjustable bell intervals (1-15 minutes)
		- Custom audio file selection (MP3)
		- 1-minute preparation countdown
		- Deep bell sounds at intervals and completion
		- Ava voice welcome message
		- Background audio looping
		- Session history and statistics
		- Pause/resume/stop controls
		- Settings persistence
